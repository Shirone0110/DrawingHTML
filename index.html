Drawing HTML

In this chapter we learned how to create page elements that contains data from a particular source. This page will explain how select(), append(), text(), and style() commands work.

select()

The select() function takes a CSS selector and returns the first element matches that selector. For example: 
                <h1>Header</h1>
                <p id = "p1">Paragraph 1</p>
                <p id = "p2">Paragraph 2</p>
d3.select("h1") will return h1 element Header, d3.select("#h2") will return p element Paragraph 2, d3.select("p" will return p element Paragraph 1.

append()

The append() function takes an element type and add it to the end of whatever it's acting on. For example 
                var body = d3.select("body");
                body.append("p");
will turn 
                <body>
                    <h1>Header</h1>
                </body>
into 
                <body>
                    <h1>Header</h1>
                    <p></p>
                </body>

text()

The text() function takes a string and set to the value between the opening and closing tags of whatever it's acting on. For example 
                var p = d3.select("p");
                p.text("Hello World");
will turn 
                <p>Paragraph</p>
into 
                <p>Hello World</p>
When saying "set the value between the opening and closing tags", it means replace everything with the new value, not adding more to it. For example 
                var p = d3.select("p");
                p.text("Hello World");
                p.text("Hi");
will result in 
                <p>Hi</p>

style()

The style() function sets CSS properties to whatever it's acting on. For example 
                d3.select("a").style("text-decoration", "none");
will get rid of the underline in links.
The elements in the Example section above is the result of 
                var body = d3.select("#example");
                body.append("h3").text("Example");
                body.append("p").text("This paragraph is created by javascript");
                body.append("p").text("This too").style("color", "red");
